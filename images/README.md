# Lokale Pflanzenbilder

Bilder hier ablegen als: `{plant-id}.jpg` (oder .png / .webp)

Dann in js/taxon-ids.js beim entsprechenden Eintrag ergänzen:
  localImage: "images/{plant-id}.jpg"

Beispiel:
  "corylus-avellana-contorta": { ..., localImage: "images/corylus-avellana-contorta.jpg" }

Bildquellen (frei verwendbar):
- Wikimedia Commons: https://commons.wikimedia.org
- Pixabay: https://pixabay.com
- Unsplash: https://unsplash.com
